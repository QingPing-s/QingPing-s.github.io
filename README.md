# AI Agent 工程博客

Personal portfolio and technical blog built with Astro and designed for GitHub Pages.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Update `astro.config.mjs` with your GitHub Pages domain, then push to GitHub. The workflow in `.github/workflows/deploy-blog.yml` builds this subproject and publishes it to GitHub Pages.
